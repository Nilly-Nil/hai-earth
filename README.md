# h/ai static site

A minimal static landing page for H/AI.

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## Files

- `index.html` — page structure/content
- `styles.css` — visual design
- `script.js` — rotating device display

## Deploy options

- GitHub Pages
- Cloudflare Pages
- Vercel
- Netlify
