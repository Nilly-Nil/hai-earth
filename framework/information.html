H/AI Primitive Language

Volume II
Information Primitives

Information primitives describe how meaning is acquired, represented, transformed, preserved, and communicated across Human–AI systems.

Draft v0.1Volume IIInformation Flow

Overview

Information primitives model how meaning moves through a Human–AI system independently of temporal traversal. They separate the channel used to acquire or deliver information from the representation used to encode it.

This distinction becomes critical in multilingual, multimodal, and accessibility-aware systems where the same intent may move through speech, text, vision, gesture, captions, internal commands, and external action systems simultaneously.

Primitive Definitions

Modality

The sensory or interaction channel through which information is acquired or delivered.

Representation

The encoding used to express information within or across modalities.

Intent

The semantic objective preserved throughout interpretation, transformation, and action.

Transformation

Conversion between modalities or representations while preserving intent.

Context

The situational interpretation attached to information within a specific interaction.

Modality Examples

Modality defines the channel, not the language.

  • Speech
  • Text
  • Vision
  • Gesture
  • Touch
  • Audio
  • Video

Representation Examples

Representation defines the encoding used to express meaning.

  • English
  • Japanese
  • Italian
  • ASL
  • Braille
  • JSON
  • Emoji
  • System command language

Language Roles

Language is not a single primitive. It appears as a Representation role attached to different system positions.

  • Input language: the language used to instruct the system through any modality.
  • Spoken language: the language used in speech input or speech output.
  • Displayed language: the language used in visual interface content.
  • Caption language: the language used for translation or accessibility display.
  • System language: the internal representation used for commands, triggers, intent, or data exchange.
  • Output language: the language used to communicate results to an external person or system.

Information Principle

Meaning may move across modalities and representations without changing the underlying intent. Human–AI systems must preserve intent while transforming information for users, interfaces, internal reasoning, and external action.
